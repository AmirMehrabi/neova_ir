# Header, Search & Sidebar Improvements

## Changes Overview

### 1. Compact Sidebar
- Remove workspace avatar/initial circle
- Reduce padding: `px-3 py-2` → `px-2.5 py-1.5`
- Reduce item height, text size
- Just name + project count, no icon

### 2. Search Input (replaces workspace name in header)
- Full-width search input at the start of the navbar
- Debounce: 300ms before API call
- Searches: workspace names, project names, task titles
- Results dropdown grouped by type (فضاهای کاری / پروژه‌ها / وظایف)
- Recent searches stored in localStorage, shown when input is focused with no query
- Click result → navigate to that item
- Keyboard navigation (up/down arrows, enter to select)

**Backend:**
- New route: `GET /dashboard/search?q=...`
- New method: `DashboardController::search()`
- Queries: workspaces (user's), projects (in user's workspaces), tasks (in user's projects)
- Returns JSON array of {type, name, url, subtitle}

### 3. User Dropdown (replaces avatar+name+logout)
- Click avatar → dropdown with: user name, email/phone, separator, logout button
- Dropdown closes on click outside or Escape
- Clean, minimal dropdown design

### 4. "پروژه جدید" Button
- SVG plus icon + text, same as now but ensure icon is SVG not text
